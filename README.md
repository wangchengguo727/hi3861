# **基于HI3861的多环境感知与AI交互系统项目拆解**

## **一、 项目概述**

本项目以 **“环境感知 \- 智能分析 \- 主动服务”** 为核心逻辑，采用 **“HI3861 \+ 鸿蒙APP \+ 云端大模型”** 的三层架构，聚焦鸿蒙生态下的嵌入式开发、多传感器融合与 AI 智能交互能力的培养。

### **1.1 系统架构**

* **感知层 (HI3861):** 连接多种环境传感器实时采集数据。  
* **协同层 (HarmonyOS):** 借助鸿蒙分布式能力实现跨设备协同。  
* **智能层 (AI Model):** 通过 MCP 协议接入大模型进行智能分析。  
* **展示层 (DevEco Studio):** 开发图形化界面提供直观展示和智能控制。

### **1.2 应用场景**

1. **家庭环境智能化管理：** 实时监测室内环境质量，自动调节家居设备。  
2. **安全防护与预警：** 烟雾、门窗状态监测，异常情况及时预警。  
3. **AI 辅助生活决策：** 借助大模型分析习惯，提供个性化建议。

## **二、 核心功能模块**

* **多环境参数精准感知：** HI3861 采集温湿度、光照、烟雾、空气质量等，采用滤波算法确保数据准确。  
* **鸿蒙分布式协同控制：** 跨手机、平板、智慧屏的数据同步与无缝操作体验。  
* **大模型智能分析与交互：** 通过 MCP 协议实现自然语言交互、智能场景推荐及异常行为预警。  
* **可视化数据呈现与场景管理：** 实时展示环境趋势，支持睡眠、会客、离家等模式切换。  
* **自适应环境调节系统：** 自动调节空调、开窗通风或光照亮度，平衡舒适度与能耗。  
* **安全防护与健康关怀：** 集成安全监测，为老人和儿童提供健康生活建议。

### **第三阶段：系统集成与算法进阶 (联合攻关)**

**目标：** 软硬结合，算法落地，从“能动”变“智能”。

* **任务八：APP 落地 (软件组)**  
  * 实现 MQTT 连接，将数据上传至华为云并接收下发指令。  
  * *关键点：* 连接稳定性与收发流转。  
* **任务九：拿下 HI3861 (电控组)**  
  * 编写 MQTT 连接函数，实现数据上报与解析执行下发指令。  
  * *关键点：* 多传感器数据融合。  
* **任务十：阶级跨越 (网络组)**  
  * 整合 MCP 与 APP、HI3861 的交互。例如：询问“现在温度如何？”，AI 回复具体数值及建议。  
* **任务十一：实地调试 (全员)**  
  * 系统独立供电，全流程跑通（环境感知 \-\> 云端分析 \-\> APP展示/AI交互）。  
  * *关键点：* 供电管理（12V转5V共地）、线程管理、Debug能力。
